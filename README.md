# 🗃️ 人生档案 (Life Archive)

这是我的个人信息档案系统，记录我一生的重要信息。

---

## 📖 如何使用

### 对于我自己
- 需要查找信息时，直接浏览对应目录
- 新信息放入 `⚙️ 工具区/_inbox 📥临时收集箱/`，定期整理归档
- 敏感信息存放在 `⚙️ 工具区/_private 🔒高敏感信息/`

### 对于 AI
1. 首先阅读 `SUMMARY.md` 了解档案主人
2. 查看 `⚙️ 工具区/_ai-guide 🤖AI使用指南/` 了解使用规范
3. 按需查阅具体目录

## 📝 使用说明

### 信息录入
1. 随时记录放入 `_inbox 📥临时收集箱`
2. 定期整理归档到对应目录
3. 重要变更记录在 `CHANGELOG.md`

### 标签系统
文件头部可使用 YAML front matter 添加标签：
```yaml
---
tags: [重要, 职业, 2024]
date: 2024-12-10
---
```

### 隐私保护
- 敏感信息放入 `_private 🔒高敏感信息/`
- 该目录已加入 `.gitignore`
