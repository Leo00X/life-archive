# 🔄 定期回顾

此目录存放周期性的回顾记录。

---

## 📁 子目录

| 目录 | 回顾周期 | 建议时间 |
|------|----------|----------|
| `weekly/` | 每周 | 周日晚上 |
| `monthly/` | 每月 | 月末 |
| `yearly/` | 每年 | 年末 |

---

## 回顾模板

### 周回顾
- 本周完成了什么？
- 有什么收获？
- 下周计划？

### 月回顾
- 本月目标完成情况？
- 有什么成长？
- 需要调整什么？

### 年回顾
- 年度目标完成情况？
- 最大的成就？
- 最大的挑战？
- 明年的方向？
