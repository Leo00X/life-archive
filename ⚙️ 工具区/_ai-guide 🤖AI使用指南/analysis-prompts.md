# 常用分析 Prompts

收集用于分析用户信息的常用 prompt。

---

## 🧠 性格分析

### 综合性格画像
```
请根据以下测评结果，为我生成一份综合性格画像：
- MBTI: [结果]
- 大五人格: [结果]
- 九型人格: [结果]
- 盖洛普优势: [结果]

请分析：
1. 核心性格特征
2. 优势与潜力
3. 可能的挑战
4. 适合的工作环境
5. 人际关系建议
```

### 成长建议
```
基于我的性格档案，请给出个人成长建议：
1. 需要发展的能力
2. 可能的盲点
3. 具体可行的改进步骤
```

## 💼 职业规划

### 职业方向分析
```
根据我的：
- 技能: 见 skills/
- 经历: 见 career/
- 性格: 见 personality/
- 兴趣: 见 preferences/hobbies.md

请分析适合我的职业方向和发展路径。
```

---

## 🎯 目标回顾

### 年度回顾
```
请帮我回顾今年的目标完成情况：
1. 已完成的目标
2. 未完成的目标及原因分析
3. 对明年的建议
```

---

## 🔍 模式识别

### 发现行为模式
```
请分析我在以下方面的行为模式：
- 决策记录: decisions/
- 反思日志: reflections/
- 对话历史: conversations/

识别重复出现的模式和潜在的改进点。
```
